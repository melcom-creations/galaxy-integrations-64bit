# galaxy-integrations-64bit

Community integrations for the native 64-bit version of GOG Galaxy 2.1+.

This repository serves as the future home of several community integrations that have been updated, modernized, and adapted for the current 64-bit GOG Galaxy client.

A number of integrations are already in active development. Some have progressed beyond the internal testing stage and are currently being evaluated by a small group of beta testers.

The purpose of this repository is to provide information about the project and its ongoing development. Public releases are being prepared and may become available at any time as testing and validation are completed.

The goal is to preserve and extend community integrations that are no longer actively maintained while ensuring compatibility with modern versions of GOG Galaxy.

More information, release packages, documentation, and supported integrations will be published here soon.

Stay tuned.
